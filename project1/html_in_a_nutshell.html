Thinking Like a Programmer ...

There are five key ways to think like a programmer:

  • Procedural Thinking which involves creating simple instructions for a computer to follow.
  • Abstract Thinking which involves finding commonalities among different things.
  • Systems Thinking which may sound intimidating, but this involves breaking down a large problem into smaller tasks.
  • Technological Empathy involves being able to understand a computer's actions.
  • Debugging which allows programmers to fix problems.
